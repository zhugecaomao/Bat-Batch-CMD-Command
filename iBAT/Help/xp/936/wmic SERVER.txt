
SERVER - 服务器信息管理。

提示: BNF 的别名用法。
(<alias> [WMIObject] | <alias> [<path where>] | [<alias>] <path where>) [<verb clause>].

用法:

SERVER ASSOC [<format specifier>]
SERVER CREATE <assign list>
SERVER DELETE
SERVER GET [<property list>] [<get switches>]
SERVER LIST [<list format>] [<list switches>]
