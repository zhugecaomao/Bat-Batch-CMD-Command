
下列指令有效:

此上下文中的命令:
?              - 显示命令列表。
add            - 在一个表格中添加一个配置项。
delete         - 从一个表格中删除一个配置项。
dump           - 显示一个配置脚本。
export         - 将局域网配置文件保存到 XML 文件。
help           - 显示命令列表。
reconnect      - 在一个接口重新连接。
set            - 在接口上配置设置。
show           - 显示信息。

若需要命令的更多帮助信息，请键入命令，
后面跟 ?。
