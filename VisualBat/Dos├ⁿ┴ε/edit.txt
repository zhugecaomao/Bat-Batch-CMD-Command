Edit



启动 MS-DOS 编辑器，它可创建和更改 ASCII 文本文件。如果不指定参数，edit 就启动 MS-DOS 编辑器。


语法



edit [[Drive:][Path] FileName] [/b] [/g] [/h] [/nohi]


参数



	[Drive:][Path] FileName

	指定 ASCII 文本文件的位置和名称。如果文件不存在，MS-DOS 编辑器将创建该文件。如果文件存在，MS-DOS 编辑器将打开该文件并在屏幕上显示其内容。

	/b

	以黑白方式显示 MS-DOS 编辑器。如果单色监视器不能正确显示 MS-DOS 编辑器，请使用此命令行选项。

	/g

	使用 CGA 监视器可能的最快屏幕更新速度。

	/h

	显示当前监视器可以显示的最大行数。

	/nohi

	允许您在 8 色监视器上使用 MS-DOS 编辑器。一般来说，Windows XP 使用 16 色。

/?在命令提示符显示帮助。



注释


XOX
 默认情况下，有些监视器可能不支持显示快捷键。如果您的监视器没有显示快捷键，请使用 /b（对于 CGA 监视器）和 /nohi（对于不支持粗体字符的系统）。
